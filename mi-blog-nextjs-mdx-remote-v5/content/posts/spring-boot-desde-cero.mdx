# Spring Boot desde cero

Spring Boot facilita el desarrollo de aplicaciones backend con Java.

## Temas base

1. Inyección de dependencias
2. REST API
3. JPA / Hibernate
4. Seguridad con JWT

<Alert type="warning">
Cuando armes una API real, define desde el inicio una estructura clara por capas: controller, service, repository, dto y mapper.
</Alert>
