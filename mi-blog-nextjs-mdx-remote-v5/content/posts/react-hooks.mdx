# React Hooks

Los hooks permiten trabajar con estado y efectos en componentes funcionales.

## Hooks comunes

- useState
- useEffect
- useMemo
- useCallback

## Ejemplo de useState

```jsx
import { useState } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)

  return (
    <button onClick={() => setCount(count + 1)}>
      {count}
    </button>
  )
}
```
