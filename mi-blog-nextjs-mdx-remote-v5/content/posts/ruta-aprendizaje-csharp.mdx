## **ASP.NET Core Web API**
Es un framework de Microsoft para crear servicios web / APIs HTTP

Con él puedes crear endpoints como estos:

GET /api/clientes  
POST /api/clientes  
PUT /api/clientes/5  
DELETE /api/clientes/5  

Es utilizad para el CRUO, autenticacion JWT, Conexion a Base de datos, Logica del Negocio y consumir  
API externas .
