# PostgreSQL para backend

PostgreSQL es una excelente opción para APIs REST y sistemas empresariales.

## Puntos que conviene dominar

- tipos de datos
- índices
- constraints
- funciones
- vistas
- optimización de consultas

## Ejemplo de tabla

```sql
CREATE TABLE customer (
  id BIGSERIAL PRIMARY KEY,
  full_name VARCHAR(150) NOT NULL,
  email VARCHAR(120) UNIQUE
);
```
