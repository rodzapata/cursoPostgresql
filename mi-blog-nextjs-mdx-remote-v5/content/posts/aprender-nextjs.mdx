# Aprender Next.js desde cero

Next.js es un framework basado en React para construir aplicaciones modernas.

## ¿Qué es Next.js?

Next.js agrega capacidades como renderizado híbrido, rutas basadas en archivos y optimizaciones de rendimiento.

<Alert type="success">
Si vienes de React, Next.js te aporta rutas, layouts, rendering del lado del servidor y mejor estructura para proyectos grandes.
</Alert>

## Ejemplo de ruta

```jsx
export default function Page() {
  return <h1>Hola desde Next.js</h1>
}
```
